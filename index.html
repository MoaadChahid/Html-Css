Html&Css

HTML (HyperText Markup Language)

HTML (HyperText Markup Language) è un linguaggio usato per strutturare il contenuto di una pagina web.

Serve a definire cosa sono gli elementi presenti in una pagina (come titoli, paragrafi, liste, collegamenti, ecc.). Non è un linguaggio di programmazione, ma un linguaggio di marcatura: organizza le informazioni in modo gerarchico e logico.

CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) è un linguaggio usato per definire l’aspetto visivo degli elementi strutturati con HTML.

ermette di specificare colori, spaziature, dimensioni, font, posizioni e altri aspetti grafici.
